# mtg-proto

Contract definitions for the MTG collection platform. OpenAPI specs for external REST interfaces, Protobuf for internal gRPC.

## Structure

```
mtg-proto/
├── openapi/v1/          # OpenAPI 3.0 specs (REST)
│   ├── cards.yaml
│   ├── collection.yaml
│   ├── users.yaml
│   └── common.yaml      # shared schemas (pagination, errors, health)
├── proto/               # Protobuf definitions (gRPC)
│   ├── cards/v1/
│   ├── collection/v1/
│   └── users/v1/
└── gen/                 # Generated code (do not edit)
    ├── openapi/         # generated by oapi-codegen
    └── proto/           # generated by buf/protoc
```

## Prerequisites

- [Go 1.22+](https://go.dev/dl/)
- [Buf CLI](https://buf.build/docs/installation)
- [oapi-codegen](https://github.com/oapi-codegen/oapi-codegen)

```bash
go install github.com/oapi-codegen/oapi-codegen/v2/cmd/oapi-codegen@latest
go install github.com/bufbuild/buf/cmd/buf@latest
```

## Usage

```bash
# Generate all code (proto + openapi)
make generate

# Generate only protobuf
make proto

# Generate only OpenAPI
make openapi

# Lint protobuf definitions
make lint

# Remove generated files
make clean
```

## Consuming this module

```go
import (
    cardsv1 "github.com/eljayman/mtg-proto/gen/proto/cards/v1"
    collectionv1 "github.com/eljayman/mtg-proto/gen/proto/collection/v1"
    usersv1 "github.com/eljayman/mtg-proto/gen/proto/users/v1"
)
```
